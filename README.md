# Maze Game

Maze Game is a 3D interactive game developed in Unity using C#. The player navigates through a maze while the game handles movement, collision detection, and win/loss conditions.

## Technologies
- Unity
- C#

## Features
- 3D maze environment
- Player movement mechanics
- Collision detection
- Win and loss conditions
- Scene management
- Interactive gameplay

## Project Structure
- Unity project files
- Scripts
- Assets
- Scenes

## Run Locally
Open the project in Unity and run the main scene from the Unity Editor.
